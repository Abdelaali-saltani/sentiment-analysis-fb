About the Project

Sentiment Analysis of Facebook Comments using Machine Learning

Project Goal

This project demonstrates a complete Natural Language Processing (NLP) pipeline for classifying social media comments into three sentiment categories: Positive, Negative, and Neutral. The system uses machine learning to automatically analyze text and predict sentiment with 98.14% accuracy.

Key Features

Machine Learning

Logistic Regression classifier with TF-IDF feature extraction

3-Class Classification

Predicts Positive, Negative, and Neutral sentiments

Web Interface

Modern Flask-based UI with real-time analysis

High Accuracy

98.14% accuracy on 10,000+ sample dataset

Fast Processing

Instant predictions with confidence scores

Responsive Design

Works on desktop, tablet, and mobile devices

Technology Stack

Backend

Python Flask scikit-learn NLTK

Machine Learning

Logistic Regression TF-IDF Text Preprocessing

Frontend

HTML5 CSS3 JavaScript Chart.js

NLP Pipeline

From raw text to sentiment prediction in 4 powerful steps

01

Clean and normalize raw text through lowercasing, URL removal, punctuation elimination, and stopword filtering

Cleaning Normalization
02

TF-IDF Vectorization
03
